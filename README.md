# Lucel Power Design

Gerador de slides profissionais com estilos de marcas famosas.

## Como subir no Render

### 1. Criar repositório no GitHub
1. Acesse github.com → clique em **+** → **New repository**
2. Nome: `lucel-power-design`
3. Deixe **Private**
4. Clique em **Create repository**
5. Faça upload dos arquivos: `server.py`, `index.html`, `requirements.txt`

### 2. Subir no Render
1. Acesse render.com → **New +** → **Web Service**
2. Conecte o repositório `lucel-power-design`
3. Configure:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `python server.py`
   - **Instance Type:** Free
4. Clique em **Create Web Service**

Pronto! Em 2 minutos estará no ar.

## Como usar
1. Acesse a URL gerada pelo Render
2. Digite o tema da apresentação
3. Escolha o estilo de marca
4. Informe os pontos principais
5. Clique em **Gerar apresentação**
6. Baixe o HTML ou apresente direto na tela
